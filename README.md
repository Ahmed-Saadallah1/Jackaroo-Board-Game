Jackaroo Board Game
A fully playable digital implementation of the Jackaroo board game built in Java using OOP principles and JavaFX for the GUI.
Features

Complete game engine with all card actions and marble movement logic
JavaFX interactive GUI with real-time board updates
1 human player vs 3 AI (CPU) players
Full exception handling for all invalid moves
Trap cells, Safe Zones, and special cards (King, Jack, Seven, Burner, Saver)
Keyboard shortcuts for marble fielding

Tech Stack

Java
JavaFX
OOP (Inheritance, Interfaces, Abstract Classes, Exception Handling)

How to Run

Clone the repository
Open in any Java IDE (IntelliJ / Eclipse)
Make sure JavaFX is configured
Run the Game class
